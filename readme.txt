Theme Detector plugin help you to detect which wordpress theme is used by wordpress site. 

== Description ==

This tool help  visitors to scan competitors websites to know what wordpress theme they are using.
The results will contain the theme name, version and description.
 
This plugin is FREE of charge and can be implemented on any page or post on your site.
If you are looking for something that will keep your users engaged and interested with your site, look no further. 
Implementing this free tool will increase your users' enganement while improving your site's overall experience. 
A huge SEO plus for you!

== Installation ==

The best practice to install the plugin is:

1. Download the plugin
2. In your admin panel, go to Plugins -> Add New
3. Click "Upload Plugin" and choose the Zip file you downloaded
4. After the plugin is uploaded click "Activate plugin"

If you rather do things manually:

1. Upload 'theme-detector' folder to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress

== Implementing ==

In order to implement the tool on your page or post, 
you have to just paste <code>[detector_searchform]</code> this code into your webpage for searchform.
and find output by using <code>[detector_layout]</code> code in your webpage.
After installation, an explanation page will be added under the plugins tab in the admin area.
